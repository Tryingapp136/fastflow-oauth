FastFlow Security Policy

Overview

At FastFlow, we take the security of your personal data seriously. This policy outlines our security measures and how we handle security vulnerabilities.

Data Protection

The FastFlow app implements the following security measures:

  • Data is stored locally on your device whenever possible
  • Sensitive data is encrypted at rest
  • OAuth 2.0 is used for third-party integrations (like Withings)
  • No data is shared with third parties without your consent

OAuth Security

Our Withings integration uses industry-standard OAuth 2.0 authorization flow:

  • Authentication occurs directly on Withings' servers
  • FastFlow never sees or stores your Withings password
  • Access tokens are encrypted and stored securely
  • You can revoke access at any time from the app's settings

Vulnerability Reporting

If you discover a security vulnerability in FastFlow, please report it to our security team at security@fastflow-app.com.

Please include:

  • A description of the vulnerability
  • Steps to reproduce the issue
  • Potential impact of the vulnerability

Contact

For any security-related questions or concerns, please contact us at security@fastflow-app.com.
